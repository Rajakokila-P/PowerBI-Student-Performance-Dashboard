# Student Performance Dashboard

## Project Overview

This Power BI project analyses student performance data using an interactive dashboard. The dashboard focuses on maths, reading, and writing scores, and helps identify patterns in student performance.

## Dashboard Features

- Average Math Score KPI
- Average Reading Score KPI
- Average Writing Score KPI
- Gender Analysis
- Reading vs Writing Score Comparison
- Student Performance Distribution

## Tools Used

- Power BI
- Data Analysis
- Data Visualisation
- GitHub

## Project File

- `StudentPerformanceDashboard.pbix` - Power BI dashboard file

The Power BI file contains the dashboard report. The original dataset can be added later if available.
